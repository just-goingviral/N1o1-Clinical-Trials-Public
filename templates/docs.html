Nitrite Dynamics API

Clinical trial simulator for nitric oxide (NO) metabolism

API Documentation

This API provides endpoints for simulating nitrite (NO2-) dynamics after supplementation.

GET /patients

Get a list of all patients in the clinical trial.

Response

{
  "status": "success",
  "data": [
    {
      "id": 1,
      "name": "Patient Name",
      "age": 65,
      "weight_kg": 75.5,
      "baseline_no2": 0.15,
      "notes": "Patient notes",
      "created_at": "2025-04-18T01:00:00"
    },
    ...
  ]
}

POST /api/simulate

Run a nitrite dynamics simulation.

Request Body

{
  "patient_id": 1,  // optional, if not provided uses default values
  "model_type": "1-compartment PK",
  "dose": 30.0  // mg
}

Response

{
  "status": "success",
  "data": {
    "time_points": [0, 10, 20, ...],
    "nitrite_levels": [0.2, 0.8, 1.5, ...],
    "parameters": {
      "baseline": 0.2,
      "peak": 4.0,
      "peak_time": 30,
      "half_life": 0.5,
      "dose": 30.0,
      "age": 65,
      "weight": 75.5,
      "model_type": "1-compartment PK"
    }
  }
}

About Nitrite Dynamics

This application is designed to support clinical trials studying nitric oxide metabolism. It provides tools for:

  • Simulating nitrite curves after supplementation
  • Optimizing dosing schedules
  • Visualizing therapeutic windows
  • Patient data management

Created in collaboration with Dr. Nathan S. Bryan, a globally recognized expert in nitric oxide biochemistry.
